# CLAUDE.md

## Project Overview
Dotfiles repo for a minimal, containerized dev environment.

## Core Tools
- **Vim** (not Neovim) - keep it simple; TypeScript and Go support WIP
- **tmux** - simple layout, copy/paste
- **bash** - default shell
- **gitconfig** - git aliases
- **lynx** - browser

## Conventions
- Prefer simplicity over features
- No Neovim, no bloat
- Containerized workflow (podman/distrobox planned)

## Directory Structure
```
~/Repos
  - <domain>.com
    - <github-user>/
      - <repo>/
```
